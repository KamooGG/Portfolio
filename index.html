Acerca de mi

Soy un apasionado desarrollador web con experiencia sólida en la creación de interfaces visuales cautivadoras y funcionales.

Mi enfoque se centra en la maquetación pixel perfect, utilizando tecnologías modernas para ofrecer experiencias de usuario excepcionales.

Para mi trabajo uso HTML, CSS, Javascript; pero también tengo experiencia usando frameworks como Angular 17+, SASS y Stencil. Tengo experiencia en manejo de bases de datos y conexiones con APIs
